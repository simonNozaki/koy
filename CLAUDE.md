# CLAUDE.md

## 言語ルール

- **仕様・スキル・ドキュメント（`specs/`、`.specify/`、`README.md` など）**: 日本語で記述する
- **コード・コメント（`.kt`、`.kts` など）**: 英語で記述する

## コミットメッセージ

コミットメッセージは **Semantic Commit Message** で統一する。

```
<type>: <subject>
```

| type | 用途 |
|---|---|
| `feat` | 新機能の追加 |
| `fix` | バグ修正 |
| `refactor` | 動作を変えないリファクタリング |
| `docs` | ドキュメントのみの変更 |
| `test` | テストの追加・修正 |
| `chore` | ビルド設定・依存関係など雑務 |
| `perf` | パフォーマンス改善 |

- `subject` は英語・命令形・小文字始まり（例: `add for-in step support`）
- 1行目は 72 文字以内に収める
- 本文が必要な場合は 1行空けて追記する
